
# My Bird Life List

Bird Life List is a open-source Pyside-6 project, built for personal use, which i have decided to share! Instead of using an excel sheet to save my birdwatching life list, I have created a 2000s style UI, that looks like an old computer program.

## Features

#### Import/Export CSVs
Import your current life list from a CSV file, the loader allows you to select which column refers to which field inside of My Bird Life List. 

#### Add Images / Locations
Each bird species can have a location and image linked to it, which can be viewed by that species' record details. Open source mapping used, which is based on the location field of the species. Photos are small, and match the theme of the application. 

#### Multiple Lists
Users can have multiple life lists from countries around the world. 

#### Additional functionality
Contains all the necessary QOL features you should require for birding. Add/delete records, search birds and save changes made.
